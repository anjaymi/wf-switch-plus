# WF Switch Plus 说明文

## 项目定位

WF Switch Plus 是 `wf-switch` 的伴生增强插件，不替代原版插件。它通过独立控制台和伴生桥，补充持续对话、附件上传、Token 统计、模型成本估算、账号总览和快捷控制能力。

## 安装步骤

1. 下载 `releases/wf-switch-plus-0.3.6.vsix`。
2. 打开 Windsurf 或 VS Code。
3. 进入扩展面板。
4. 选择 `Install from VSIX...`。
5. 安装后重载窗口。

命令行安装：

```powershell
code --install-extension .\releases\wf-switch-plus-0.3.6.vsix
```

## 持续对话功能

持续对话由以下模块组成：

- `src/continueSupport.js`：弹窗、HTTP 服务、提交处理。
- `src/continueDialogHtml.js`：继续对话 Webview HTML。
- `src/continueAttachments.js`：图片粘贴、附件和结构化请求。
- `src/continueRegistry.js`：窗口注册和端口文件。
- `scripts/wf-switch-continue.ps1`：对话结束时触发弹窗的 PowerShell 脚本。

如果点击“继续对话 / 上传附件”后弹窗不出现，优先确认已安装 `0.3.3` 或更新版本。`0.3.3` 已修复旧面板引用失效后不重建的问题。

## 0.3.3 安装引导

`0.3.3` 增加首次安装/升级弹窗，引导用户安装持续对话规则与本地 HTTP。

弹窗按钮包括：

- `安装规则与 HTTP`：执行完整安装流程。
- `打开控制台`：打开 WF 增强控制台，用户可手动点安装规则与 HTTP。
- `稍后`：本版本不再继续弹出。
- `不再提示`：永久关闭该引导。

本地 HTTP 服务仍会随扩展启动，用于兼容已安装规则的用户；全局规则不再静默重写，必须由用户点击安装或更新。

## 0.3.4 质检与自动修复

`0.3.4` 增加持续对话质检功能。质检会检查脚本、全局规则、端口文件、HTTP 连通性和窗口注册。

可通过控制台“质检/修复”按钮或命令 `WF 增强: 持续对话质检/自动修复` 手动触发。后台检测到异常时，也会提示用户自动修复。

## 0.3.5 Token 趋势卡修复

`0.3.5` 修复趋势卡右上角显示 `0%` 的问题。该位置现在显示真实节省率 `省 X%`，并和 5 秒实时刷新保持一致。

## 0.3.6 接入 Windsurf 官方模型价格表

`0.3.6` 不再使用硬编码 blended 单价，改为直接读取 Windsurf 本机缓存的官方 `ClientModelConfig` 并解析其 `ModelDimension`（Input / Cached input / Output 三段），得到与客户端完全一致的 $/1M 价目。

- **数据来源**：`state.vscdb` → `ItemTable.windsurfConfigurations`（protobuf 二进制，本机缓存，不发起任何网络请求）。
- **覆盖范围**：120+ 款 Cascade 模型，包含 `creditMultiplier`（例如 Claude Opus 4.7 Medium 为 40x、GPT-5.5 High 为 24x），与硬编码表相比可修正数倍误差。
- **启动行为**：扩展激活时自动刷新一次，解析失败则静默回落旧估算值。
- **手动刷新**：控制台 `伴生桥与工具` 卡片新增 `同步官方价格` 按钮；命令面板搜索 `WF 增强: 同步 Windsurf 官方模型价格`。
- **面板展示**：控制台 Hero 区新增一行 `官方价 · <模型> · Input / Cached / Output / credit`，直接引用官方价目。若本机还未缓存到官方数据，则标注为 `估算价 · blended …`。
- **新增模块**：`src/protobuf.js`（通用 wire-format 解码）、`src/windsurfPricing.js`（ClientModelConfig 解析）、`src/windsurfState.js`（只读本机 SQLite）、`src/vendor/sql/`（纯 wasm SQLite 依赖，约 0.7 MB）。

## Token 与模型统计

插件会根据 Windsurf 当前模型状态、手动模型覆盖值和价格配置估算 Token 成本。若自动模型识别不准确，可设置：

```text
wfSwitchPlus.currentModelOverride
```

示例：

```text
claude-opus-4-7-medium
gpt-5.5-high
```

## 账号与桥接

账号总览依赖原版插件的共享状态和伴生桥。常用能力包括：

- 查看账号额度。
- 导出账号 Token。
- 按日额度智能切号。
- 调用原版刷新、切号、激活和设置命令。

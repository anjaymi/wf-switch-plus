# WF Switch Plus 说明文

## 项目定位

WF Switch Plus 是 `wf-switch` 的伴生增强插件，不替代原版插件。它通过独立控制台和伴生桥，补充持续对话、附件上传、Token 统计、模型成本估算、账号总览和快捷控制能力。

- 安装包：`releases/wf-switch-plus-0.3.2.vsix`
- GitHub Pages：`https://anjaymi.github.io/wf-switch-plus/`
- GitHub 仓库：`https://github.com/anjaymi/wf-switch-plus`

## 安装步骤

1. 下载 `releases/wf-switch-plus-0.3.2.vsix`。
2. 打开 Windsurf 或 VS Code。
3. 进入扩展面板。
4. 选择 `Install from VSIX...`。
5. 安装后重载窗口。

命令行安装：

```powershell
code --install-extension .\releases\wf-switch-plus-0.3.2.vsix
```

## 持续对话功能

持续对话由以下模块组成：

- `src/continueSupport.js`：弹窗、HTTP 服务、提交处理。
- `src/continueDialogHtml.js`：继续对话 Webview HTML。
- `src/continueAttachments.js`：图片粘贴、附件和结构化请求。
- `src/continueRegistry.js`：窗口注册和端口文件。
- `scripts/wf-switch-continue.ps1`：对话结束时触发弹窗的 PowerShell 脚本。

如果点击“继续对话 / 上传附件”后弹窗不出现，优先确认已安装 `0.3.2` 或更新版本。`0.3.2` 已修复旧面板引用失效后不重建的问题。

## Token 与模型统计

插件会根据 Windsurf 当前模型状态、手动模型覆盖值和价格配置估算 Token 成本。若自动模型识别不准确，可设置：

```text
wfSwitchPlus.currentModelOverride
```

示例：

```text
claude-opus-4-7-medium
gpt-5.5-high
```

## 账号与桥接

账号总览依赖原版插件的共享状态和伴生桥。常用能力包括：

- 查看账号额度。
- 导出账号 Token。
- 按日额度智能切号。
- 调用原版刷新、切号、激活和设置命令。
